# 言語聴覚士向け絵カードアプリ

このプロジェクトは、言語聴覚士が使用する絵カードを表示するためのWebアプリケーションです。ブラウザ上で簡単に絵カードを表示し、ユーザーインタラクションを管理することができます。

## プロジェクト構成

- `src/index.html`: アプリケーションのメインHTMLファイルです。絵カードを表示するための基本的な構造が含まれています。
- `src/css/styles.css`: アプリケーションのスタイルを定義するCSSファイルです。レイアウトやデザインに関するスタイルが記述されています。
- `src/js/app.js`: アプリケーションのメインJavaScriptファイルです。絵カードの表示やユーザーインタラクションを管理するためのロジックが含まれています。
- `src/data/cards.json`: 絵カードのデータを格納したJSONファイルです。各カードの情報がJSON形式で記述されています。
- `src/audio/(audio files)`: 音声ファイルを格納するディレクトリです。絵カードに関連する音声が含まれています。
- `package.json`: npmの設定ファイルです。プロジェクトの依存関係やスクリプトがリストされています。
- `.gitignore`: Gitで無視するファイルやディレクトリを指定するファイルです。
- `README.md`: プロジェクトのドキュメントが含まれています。アプリケーションの使い方やセットアップ手順が記載されています。

## アプリケーションの立ち上げ手順

1. プロジェクトをクローンまたはダウンロードします。
2. ターミナルを開き、プロジェクトのディレクトリに移動します。
3. `index.html`ファイルをブラウザで開きます。これにより、アプリケーションが表示されます。
